# 📘 Lab Interaktif Struktur Aljabar MA3021

> Website pembelajaran interaktif untuk memahami Struktur Aljabar melalui teori, visualisasi, dan latihan.

---

## 📁 Struktur Proyek
```
📦 project-root
├── index.html
├── styles.css
├── app.js
├── 01_sets_functions.html
├── 02_elementary_number_theory.html
├── 03_finite_groups.html
├── ...
└── 10_polynomial_factorization.html
```

---

## 🧠 Konsep Desain

- Satu topik = satu halaman HTML
- Fokus pada eksplorasi, bukan hanya membaca
- Materi dibuat bertahap: teori → eksperimen → latihan

---

## 🌐 Catatan Teknis

- Tautan sumber aplikasi tetap dipertahankan
- Materi tambahan Bahasa Indonesia ditempatkan di bagian **“Pelajari lebih lanjut”** tanpa duplikasi link
- MathJax dimuat dari CDN → internet diperlukan untuk rendering rumus
- Semua interaksi visual dikendalikan oleh `app.js`

---

## 📌 Tujuan Proyek

Membantu mahasiswa melihat Struktur Aljabar bukan hanya sebagai teori abstrak, tetapi sebagai sistem yang bisa dieksplorasi dan divisualisasikan.

---

Dirancang untuk membuat matematika lebih intuitif, interaktif, dan menyenangkan untuk dipelajari.
